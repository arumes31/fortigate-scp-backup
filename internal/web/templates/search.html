Search examples & tips

The search scans the newest saved configuration of every firewall and returns each matching line together with the firewall it came from. Matching is case-insensitive; * is a wildcard that matches any characters, and everything else is matched literally (so ., -, and quotes are taken as-is). Click any example to load it into the search box.

Find a firewall by name / hostname

set hostname set hostname *FGT* set hostname *-fw01* set alias *branch*

Interfaces & addressing

edit "port1" set ip 10.* set vdom *root* set allowaccess *https*

Policies, addresses & services

config firewall policy set srcintf *wan* set service "HTTPS" set action deny config firewall vip

VPN

config vpn ipsec phase1-interface set remote-gw * set psksecret *

Admins, users & security review

config system admin set trusthost* config user local config system dns set dstaddr *all*
